 # Travesaudio

Una interfaz web sencilla para convertir archivos de audio entre diferentes formatos de manera rápida y fácil.

## Características

- Conversión entre formatos populares (MP3, WAV, OGG)
- Interfaz simple e intuitiva
- Compatible con navegadores modernos
- Diseño responsive (adaptable a móviles)

## ¿Por qué usar este convertidor?

- No necesitas instalar programas
- Uso rápido y directo desde el navegador
- Ideal para usuarios principiantes y avanzados
- Interfaz limpia y fácil de entender

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript

## Uso

1. Selecciona un archivo de audio
2. Elige el formato de salida
3. Haz clic en "Convertir"
